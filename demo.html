Layout

Primitives for structure

Containers, sections, stacks, clusters, and grid columns.

Reference

.container
Centered wrapper, max width 1200px (--container-width).
.container-lg
Centered wrapper, max width 1400px (--wide-width).
.section
Vertical padding using --space-md (5rem).
.section-tight
Smaller vertical padding using --space-sm (2rem).
.section-heading
Vertical cluster for eyebrow / title / lead, capped at --content-width.
.content-box
Wrapper with generous vertical padding (--space-lg) for standalone blocks.
.stack
Vertical flex column with --space-4 gap.
.cluster
Horizontal flex row with wrap and --space-4 gap.
.grid-2 / .grid-3 / .grid-4
Responsive CSS Grid that collapses to fewer columns on narrow screens.
.flex + .col
Flex row where each .col grows equally.
.flex-grid.cols-2/3/4
Flex-based responsive grid where children snap to fixed fractions at breakpoints.
